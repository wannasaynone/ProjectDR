# 角色互動系統

> **版本**: v1.0

## 系統目的

角色互動系統是村莊經營與戀愛養成的核心介面。玩家透過此系統與角色見面、對話、使用角色對應的村莊功能。此系統直接支撐 core-definition.md 的核心樂趣 #2（戀愛養成的情感連結）——每次互動都是加深玩家與角色關係的機會。

---

## 一、互動流程

```
Hub 主畫面
  → 點擊角色按鈕（如「村長夫人」）
    → 進入角色互動畫面
      → 自動播放對話（打字機效果）
        → 對話結束
          → 顯示功能選單（如「倉庫」「對話」）
            → 點擊功能 → 懸浮覆蓋該功能頁面
```

## 二、角色互動畫面佈局

| 區域 | 位置 | 內容 |
|------|------|------|
| 立繪區 | 左側 | 角色立繪動畫（Animator），IT 階段預留欄位 |
| 對話區 | 右上方 | 對話台詞，帶打字機效果 |
| 功能選單區 | 右下方 | 對話結束後顯示，列出角色對應的功能按鈕 |

## 三、製作人確認規則

1. **所有角色統一流程**：點擊任何角色都進入相同的互動畫面結構
2. **每次進入都播放對話**：不區分首次/重複，每次進入角色互動畫面都自動播放對話
3. **「對話」選項**：IT 階段先跑對話 log，後續串接新功能
4. **立繪動畫**：IT 階段預留 Animator 欄位，使用 placeholder 或留空
5. **功能懸浮覆蓋**：點擊功能按鈕後，功能頁面在角色互動畫面上方懸浮顯示

## 四、角色功能選單對應

| 角色 | 功能選單項目 |
|------|-------------|
| 村長夫人 | 倉庫、對話 |
| 獵人 | 探索、對話 |
| 魔女 | 藥水、對話 |
| 農女 | 農場、對話 |

## 五、打字機效果規格

- 對話台詞逐字顯示
- IT 階段使用固定速度（可配置）
- 點擊/按鍵可跳過打字機效果，直接顯示完整台詞
