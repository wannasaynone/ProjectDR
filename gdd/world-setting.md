# 世界觀設定

> **版本**: v1.0

## 系統目的

本文件定義 ProjectDR 的世界觀背景與探索系統規則。世界觀為核心循環（探索 -> 帶回物資 -> 村莊建設 -> 角色互動 -> 需要更多資源）提供敘事基礎：森林的危險性與變化性解釋了「為何需要探索」，村莊的封閉與停滯解釋了「為何需要經營」，玩家的唯一性解釋了「為何角色依賴並崇拜玩家」。三者共同支撐經營成就感與戀愛養成兩大核心樂趣。

---

## 一、世界觀概述

奇幻冒險世界。一座長年被不斷變化的森林圍住的村莊，與外界斷絕聯繫已久。村莊中生活著女性居民，維持著基本的生活機能但發展停滯。玩家作為唯一的男性，在村莊中醒來，成為唯一能進入森林又安全回來的人，由此展開經營村莊與培養感情的故事。

---

## 二、村莊設定

### 村莊狀態

- 村莊處於**有基礎設施但待開發**的狀態
- 不是廢墟，是有人在生活但發展停滯
- 經營起點是在既有基礎上擴建升級，不是從零建村
- 村莊內有**農業資源與牲畜飼養**，能維持基本生存

### 村莊安全區域

- 村莊外 3 格內為**安全區域**，魔物不會靠近
- 村民可在安全區域內進行一定程度的採集
- 玩家的價值在於能深入安全區域之外的森林，帶回更多、更稀有的資源

### 社會結構

- 玩家是村莊內目前**唯一的男人**
- 村莊中居住著女性居民（可攻略角色為多種族女性角色）

### 封閉原因

- 村莊長年被**會不斷變化的森林**圍住
- 村莊已經很久沒有與外部聯絡，形成封閉社會

---

## 三、森林設定

### 森林特性

- 森林會**不斷變化**，圍住村莊
- 變化的具體表現：每次玩家出發探索時，地圖的格子結構維持不變，但**已探索的格子會重製為未探索狀態**，**魔物配置會重新隨機生成**，**已擊敗的魔物會復活**

### 危險性

森林具有兩層危險：

1. **迷路**：出去很容易因為迷路回不來
2. **魔物**：森林中有魔物，進入森林需要戰力

---

## 四、玩家角色設定

### 身份

- **人類男性**，在村莊中醒來

### 特殊性

- 能從森林安全回來（遊戲機制保證撤離）
- 這個特殊能力讓女角崇拜
- 作為唯一能進森林又回得來的人，玩家是村莊的**唯一資源供給者**，也是村莊的**核心依賴**

### 來歷謎團

- 玩家在這裡醒來的原因是**謎團**，作為主線懸念

---

## 五、村莊操作方式

- 村莊內為**純選單移動**，玩家透過選單選擇前往不同區域/角色
- 不需要場景走動或自由移動

## 六、日夜循環概念

- **白天**：探索、擴建
- **晚上**：約會、休息

---

## 七、探索系統規則

探索系統的完整規則（共 55 條）已整合至獨立規格書，詳見：

**`gdd/exploration-system.md`**

以下為本章概述，作為快速參考。

### 系統概覽

| 子系統 | 核心設計要點 |
|---|---|
| **地圖系統** | X×Y 棋盤、多張地圖、格子揭露（WASD 自由移動+中心座標觸發）、每次出發重置探索狀態與魔物配置 |
| **玩家移動** | WASD 自由移動、碰觸魔物受傷+擊退（無硬性格子限制） |
| **物品箱系統** | 兩層計時（開啟+解鎖）、開箱不能移動、魔物獨立運作、手動拾取、離開消失、背包取捨為決策核心 |
| **背包與倉庫** | 各自獨立、出發策略取捨、回村手動存入、雙向操作 |
| **玩家數值** | HP / ATK / DEF / SPD；MP 預留待技能系統 |
| **死亡機制** | HP 歸零→時間回溯包裝、裝備保留、背包回溯快照、不消耗當日行動 |
| **戰鬥系統** | 滑鼠方向+左鍵攻擊、DMG = ATK-DEF（最小 1）、劍/槍矛/弓三種武器、快捷鍵 1~5 消耗品 |
| **魔物系統** | 獨立移動冷卻、視野追擊、格子制攻擊、不跨越探索邊界、未探索區只遊蕩 |
| **撤離機制** | 編輯器標記撤離點、配置群組隨機挑選、返回出發點或撤離點等待 6 秒 |

## 八、設計推導

以下為製作人確認的設計方向推導：

| 面向 | 推導 |
|------|------|
| **經營起點** | 在既有基礎上擴建升級，不是從零建村 |
| **玩家的村莊價值** | 唯一能進森林又回得來的人 = 唯一的資源供給者 = 村莊的核心依賴 |
| **戀愛動機來源** | 崇拜（能力）+ 依賴（資源）+ 日常相處（經營互動） |
| **日夜循環** | 白天探索擴建，晚上約會休息 |

## 九、待製作人確認事項

以下為整理過程中發現的、製作人尚未明確定義的事項：

1. ~~森林「不斷變化」的具體表現方式為何？~~ ✅ 已確認：地圖結構固定，但已探索格子重製、魔物配置重骰、已擊敗魔物復活
2. ~~村莊中除了玩家外的女性居民，在玩家到來之前如何取得森林資源維持生活？~~ ✅ 已確認：村莊內有農業與牲畜，村莊外 3 格內為安全區域可採集，魔物不會靠近
3. ~~玩家踩到魔物受傷的具體後果為何？是否有死亡機制？~~ ✅ 已確認：有 HP/ATK/DEF/SPD 數值系統，HP 歸零死亡，時間回溯（裝備保留、背包回溯至出發前快照、不浪費時間）
4. ~~玩家的攻擊方式除了滑鼠左鍵射擊外，是否有其他攻擊手段？~~ ✅ 已確認：不同武器種類改變左鍵攻擊方式（劍/槍矛/弓等）；快捷鍵 1~5 使用消耗品；技能系統待定
5. 探索時的背包容量與倉庫容量的具體數值待定義
6. ~~「不一定大的地圖」的具體規模範圍為何？~~ ✅ 已確認：各地圖獨立設定 X×Y 棋盤，支援底圖與不可生成格子區域標記，有多張地圖
7. ~~撤離點的生成時機與數量規則為何？~~ ✅ 已確認：地圖編輯時標記撤離點，多個配置群組隨機挑選，進入探索時就固定
8. core-definition.md 中「重建被摧毀的村莊」「從零開始」的措辭與本文件「有基礎設施但待開發」「擴建升級」的描述存在差異，是否需要同步更新 core-definition.md？
