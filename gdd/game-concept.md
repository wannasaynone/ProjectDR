## 基本資訊

- **遊戲名稱**: ProjectDR（暫定）
- **遊戲類型**: 村莊經營增量 + 戀愛約會養成 + 踩地雷變體探索（搜打撤）
- **世界觀**: 奇幻冒險
- **目標平台**: Steam (PC)
- **目標受眾**: 18-35 歲男性、偏好獨立遊戲與視覺小說的輕中度玩家
- **內容分級**: 成人向 (18+)
- **性別傾向**: 男性向
- **結局條件**: 暫定無限經營

## 目標受眾畫像

- **基本輪廓**: 18-35 歲男性（核心 20-30 歲），輕中度玩家，習慣 Steam PC 端，有 Visual Novel 與獨立遊戲遊玩經驗
- **核心動機**: 經營成就感（「我的努力有意義」）、情感連結（「被角色真正看見」）、安全的冒險刺激、增量式持續驅動力
- **情感體驗期待**: 與多種族女性角色（人類、精靈、獸人等）建立從陌生到深厚的情感羈絆，漸進式解鎖親密內容，成人向描寫作為高親密度的情感回報
- **參考受眾**: A Dark Room（增量經營愛好者）、HuniePop（男性向成人戀愛養成玩家）、Escape from Tarkov（搜打撤決策愛好者，取其機制概念而非硬核受眾）

## 遊戲概念

在魔物棲息的奇幻森林深處，一座與世隔絕的村莊中醒來的男人，帶領夥伴們擴建停滯已久的家園、與她們培養深厚情感，同時冒險探索不斷變化的危險森林，帶回村莊發展所需的關鍵物資。

## 核心樂趣

**主要樂趣來源**:
1. **經營成就感** — 停滯村莊在玩家手中擴建成長的可見回報，每一棟新建築都是玩家努力的具體證明
2. **戀愛養成的情感連結** — 與角色從陌生人到戀人的情感歷程，解鎖私密故事與成人內容

**附屬樂趣**:
3. **探索的緊張刺激感** — 搜打撤的風險回報抉擇（服務於經營的資源取得手段，是調味不是主菜）

**核心循環**: 村莊資源不足 → 出發探索（踩地雷式逐格揭開 + Top-down 射擊戰鬥）→ 安全撤離帶回物資 → 投入村莊建設 → 解鎖新設施與角色互動內容 → 親密度提升 → 需要更多資源 → 循環

### 核心樂趣驗證標準

> 所有設計決策都必須通過以下驗證：

1. **是否讓村莊的成長更加「可見且有意義」？**
2. **是否深化玩家對角色的情感投入？**
3. **探索的風險是否在可承受範圍且直接服務村莊成長？**
4. **是否創造了讓玩家想看下一步的驅動力？**

## 遊戲體驗關鍵字

**重建** / **羈絆** / **帶回家** / **已知與未知之間** / **恰到好處的緊張**

## 開發環境

- **遊戲引擎**: Unity 6 (6000.3.8f1)
- **開發語言**: C#
- **UI 系統**: UGUI（Canvas + GameObject）
- **版本控制**: Git
- **預設解析度**: 3840 x 2160 (4K)
- **主要框架**: KahaGameCore（GameData、GameEvent、ValueContainer、EffectProcessor、DialogueSystem、UserInterfaceSystem）、UniTask、DOTween、Addressables
