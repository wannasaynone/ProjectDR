## B14 — 玩家發問 UX（40 題清單）

### 新建檔案

**`Assets/Game/Scripts/Village/PlayerQuestionsConfigData.cs`**
- `PlayerQuestionsConfigData`（JSON DTO）、`PlayerQuestionData`（各題欄位）
- `PlayerQuestionsConfig`（不可變配置）：
  - 建構時依 `sort_order` 排序各角色題目清單
  - `GetQuestionsForCharacter(charId)` → 該角色所有題目
  - `GetUnlockedQuestions(charId, stage)` → 過濾 `unlock_affinity_stage <= stage`
  - `GetQuestion(id)` → 依 question_id 查詢
- `PlayerQuestionInfo`（不可變資料載體）：`QuestionId / CharacterId / QuestionText / ResponseText / UnlockAffinityStage / SortOrder`

**`Assets/Game/Scripts/Village/UI/PlayerQuestionsView.cs`**
- 繼承 `ViewBase`，overlay 模式（右側 w=1600）
- SerializeField：`_titleLabel / _questionListContainer / _questionRowPrefab / _answerPanel / _questionDisplayLabel / _answerText / _backToListButton / _closeButton`
- `Initialize(config, affinityMgr, dialogueMgr, characterId, charsPerSecond)` — VillageEntryPoint 注入
- `SetReturnAction(Action)` — overlay 關閉回呼
- 題目清單：`GetQuestionsForCharacter` 取全部，`CalculateAffinityStage(affinityValue / 5)` 計算當前階段
- 未解鎖題目：灰色 + 鎖定提示，`row.interactable = false`
- 點題目 → 隱藏清單面板、顯示回答面板、打字機播回答
- 返回 → 顯示清單面板；關閉 → `_returnAction?.Invoke()`
- `CalculateAffinityStage(int value) = value / 5`（IT 階段簡化）
