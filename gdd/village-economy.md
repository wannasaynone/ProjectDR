# 村莊經濟系統

> **版本**: v4.0

## 系統目的

本文件定義 ProjectDR 的基地經濟架構，涵蓋物品分類、物資流向、兩條製作路線（食物線/禮物線）、委託循環、倉庫擴建與贈禮效果系統，以及各角色的功能分工。

此系統是核心循環中「探索帶回物資→投入基地經營→基地成長解鎖新內容」環節的完整實作依據。玩家探索帶回的物資有三條出口：
1. **委託循環**：透過三位委託型角色（農女/魔女/守衛）轉化為作物／禮物／基礎物資
2. **擴建循環**：透過村長夫人擴建倉庫（+50 格/次）
3. **送禮循環**：食品與禮物直接送禮累積好感度

自 2026-04-17 委託制定案後，物資流向從「投入設施升級」轉為「委託交付 + 擴建物資 + 送禮」三條路徑。

## 一、世界觀補充說明

村莊本身有農女長期維護的農業經濟圈，村民依靠農女的農地基本上可以自給自足。玩家的農田是**額外的**：故事上是玩家主動提出要幫忙擴充糧食來源，農女才劃分出一塊空地給玩家使用。

這套設定解釋了以下兩件事：
1. 為何玩家農田需要從探索取回的種子——農女的農地已有既有品種，玩家負責帶回森林中的新品種
2. 為何村莊不會因玩家不耕種而崩潰——NPC 自給自足的農業圈始終在後台運作，玩家的農田是加成而非生存必要

## 二、物資出口總覽

探索帶回的物資有三大出口：

1. **委託循環**：透過三位委託型角色（農女/魔女/守衛）轉化為食物、禮物、基礎物資（詳見 `commission-system.md`）
2. **擴建循環**：透過村長夫人擴建倉庫（詳見 `storage-expansion.md`）
3. **送禮循環**：食品與禮物直接送禮累積好感度（詳見本文件六章）

### 兩條製作路線（委託驅動）

### 食物線（農女負責 — 耕種委託）

```
探索帶回種子
  → 進入農女 → [耕種] → 格子式工作台
    → 放入種子 → 「交給我吧」→ 倒數（現實時間）
      → 紅點 → 領取作物（food / ingredient，TBD）
        ├→ 送禮（好感度↑，食品類可送禮）
        └→ 自用（探索 BUFF，具體 BUFF 待定）
```

**註**：食材 → 食品的「料理製作」步驟，是否直接由農女的耕種委託產出食品（跳過食材層），或另設機制，為 TBD（詳見 `base-management.md` 八章 TBD#9）。

### 禮物線（魔女負責 — 煉製委託）

```
探索帶回素材
  → 進入魔女 → [煉製] → 格子式工作台
    → 放入素材 → 「交給我吧」→ 倒數
      → 紅點 → 領取藥水／寶石／金工品
        └→ 送禮（好感度↑）
```

### 基礎物資線（守衛負責 — 探索周圍委託）

```
進入守衛 → [探索周圍] → 格子式工作台
  → （空手或工具類，TBD）→ 「交給我吧」→ 倒數
    → 紅點 → 領取基礎物資（種子、初階素材）
      ├→ 作為農女耕種委託的輸入
      └→ 作為魔女煉製委託的輸入（若適用）
```

### 角色功能對應表

| 角色 | 類型 | 負責路線 | 委託輸入 | 產出 |
|------|------|---------|---------|------|
| **農女** | 委託型 | 食物線 | 種子 | 作物（食材/食品） |
| **魔女** | 委託型 | 禮物線 | 煉金素材 | 藥水、寶石、金工品 |
| **守衛** | 委託型 | 基礎物資線 | 空手（或工具，TBD） | 基礎物資（種子、初階素材） |
| **村長夫人** | 劇情型 + 功能型 | 倉庫與主線 | — | 倉庫擴建容量、主線任務引導 |

**探索 vs 守衛委託的價值分化**（2026-04-17 KA 決策後自然成立）：

| 行為 | 地理範圍 | 風險 | 產出定位 |
|---|---|---|---|
| **玩家親自探索**（Hub 探索入口） | 深入森林 | 高（戰鬥、撤離張力） | 稀有素材、特殊物資、深探獎勵 |
| **守衛探索周圍**（守衛 `[探索周圍]` 委託） | 村莊周邊 | 低（穩定產出） | 基礎物資（種子、初階素材） |

## 三、物品分類定義

所有物品在資料表中統一為「物品（Item）」，以 `type` 欄位區分種類。物品可擁有多重標籤（如番茄同時是食材和食物）。

| type 值 | 中文名稱 | 說明 |
|---------|---------|------|
| `種子` | 種子 | 從探索取得，投入農田後倒計時產出作物 |
| `食材` | 食材 | 作物收穫產出、或從探索直接取得的稀有食材；是料理製作的主要輸入材料 |
| `食品` | 食品 | 料理製作產出的成品；可送禮（好感度↑）或自用（獲得 BUFF） |
| `素材` | 素材 | 探索取得的非食用性物品，包含礦石、木材等；是煉金製作的輸入材料 |
| `禮物` | 禮物 | 煉金製作產出的寶石、金工品等；專用於送禮（好感度↑） |
| `其他` | 其他 | 不屬於上述分類的特殊物品，例如配方卷軸 |

### 送禮物品限制

**只有 `食品` 和 `禮物` 類型的物品可以送禮。** 部分物品可同時具有多重標籤（如番茄同時標記為食材和食品，可直接送禮也可作為料理原料）。

**擴充規則**：新增 type 種類前，必須先查閱資料表確認不與既有 type 衝突，避免分類混亂。

## 四、農田系統規格

### 4.1 系統定位

農田的核心功能是**種子提煉器**：將探索帶回的種子轉化為可用食材，是食物線「探索→種植→料理→送禮」中的必要中間環節。

### 4.2 運作流程

```
玩家將種子（type:種子）投入農田格
  → 倒計時開始（使用現實時間）
    → 倒計時結束，收穫作物（type:食材）
      → 作物自動留存，不會腐爛
        → 玩家手動收穫後進入倉庫
```

### 4.3 規格細節

| 項目 | 規格 | 備註 |
|------|------|------|
| 計時方式 | 現實時間 | 即使遊戲未開啟也持續計時 |
| 腐爛機制 | 無 | 收穫物不會因等待過久而消失 |
| 農田格數 | IT 階段 3 格 | 後續可透過設備升級擴充 |
| 倒計時時長 | 依種子種類不同，定義於資料表 | 各種子具體時長見物品資料表 |
| 產出作物種類 | 依種子種類對應，定義於資料表 | 具體對應關係見物品資料表 |

## 五、通用製作系統規格

### 5.1 系統定位

所有製作行為由同一套底層系統支援，差異僅在於**不同角色開放不同配方分類**。這確保未來增加新角色或新配方時，底層系統不需重寫。

### 5.2 製作流程

```
玩家在角色互動畫面選擇「製作」功能
  → 顯示該角色可用的配方清單（已解鎖配方）
    → 玩家選擇配方
      → 系統顯示所需材料（合併顯示背包 + 倉庫庫存）
        → 材料足夠 → 確認製作 → 扣除材料 → 產出成品
        → 材料不足 → 提示缺少哪些材料
```

### 5.3 材料來源讀取規則

- 製作時**背包與倉庫合併顯示**，兩者的庫存皆計入可用材料
- 扣除材料的優先順序：**先扣背包，背包不足時再扣倉庫**

### 5.4 配方解鎖機制

| 項目 | 規格 |
|------|------|
| 解鎖方式 | 收集配方卷軸（type:其他），從探索中取得 |
| 解鎖操作 | 玩家使用（或自動讀取）配方卷軸後，對應配方加入可用清單 |
| 初始配方 | 各角色有基礎配方（不需卷軸，開局即可用），具體內容定義於配方資料表 |
| 配方資料格式 | 配方表驅動（輸入材料清單 + 產出成品 + 歸屬角色分類） |

### 5.5 各路線製作內容

| 角色 | 配方分類 | 輸入 | 產出 |
|------|---------|------|------|
| **農女** | 料理製作 | 食材 → | 食品（可送禮 + 可自用 BUFF） |
| **魔女** | 煉金製作 | 素材 → | 禮物（寶石/金工品，專用於送禮） |
| **村長夫人** | 待製作人決定 | — | — |
| **守衛** | 待製作人決定 | — | — |

## 六、贈禮效果系統規格

### 6.1 系統定位

贈禮系統是角色關係推進的途徑之一。在探索主軸的設計下，好感度透過經營活動（送禮、對話、完成任務）**自然累積**，玩家不需要主動「刷好感度」。角色劇情與 HCG 作為經營里程碑獎勵，在好感度到達門檻時自動觸發。

> **開發比重備忘**：好感度系統保留所有行為，但開發比重低於探索與經營系統。

### 6.2 好感度三途徑

好感度提升有三條途徑，讓玩家與角色的關係不只靠送禮推進：

| 途徑 | 說明 |
|------|------|
| **對話問答** | 與角色對話時出現的選擇題，選對答案提升好感度。對話同時是情報收集機制——玩家從中得知角色的口味/喜好 |
| **送食物** | 送食品（type:食品）給角色，根據口味配對決定好感度增量 |
| **送禮物** | 送禮物（type:禮物）給角色，根據喜好配對決定好感度增量 |

### 6.3 贈禮流程

```
玩家在角色互動畫面選擇「送禮」功能
  → 顯示背包 + 倉庫中可送的物品清單（僅食品 + 禮物）
    → 玩家選擇物品
      → 角色根據口味/喜好觸發對應反應（對話 + 表情）
        → 好感度依物品與角色喜好等級提升
          → 達到閾值時觸發階段躍遷（CG 劇情）
```

### 6.4 角色口味喜好框架

每個角色在角色設定文件中定義以下喜好資訊：

| 喜好等級 | 說明 | 好感度變化 |
|---------|------|----------|
| 最愛 | 角色特別偏好的物品，有專屬反應對話 | 待定（最高加成） |
| 喜歡 | 普通喜好的物品，有正面反應對話 | 待定（標準加成） |
| 普通 | 無特別偏好的物品，有平淡回應 | 待定（基礎加成） |
| 不喜歡 | 角色不喜歡的物品，有負面反應對話 | 待定（可能是低加成或 0） |

> 具體好感度數值（各等級的加成量、解鎖閾值）待製作人確認。
> 各角色的具體喜好物品清單將另外撰寫獨立設計文件定義。

**重要原則**：好感度提升不能只是數字 +X。每個角色有個人的口味喜好設定，對不同禮物必須有差異化反應（不同對話與表情），讓玩家感受到「這個禮物對這個角色是特別的」。

### 6.5 好感度階段系統

- 好感度為**數值制 + 階段劃分**：以區間定義階段
- **階段躍遷**觸發重要 CG 劇情
- 每階段對話內容/語氣/親密程度不同
- 無終局設計，玩家自由遊玩

### 6.6 自用 BUFF

食品（type:食品）除了送禮外，玩家可以自用以獲得探索用 BUFF，具體 BUFF 效果待製作人確認後定義於物品資料表中。

## 七、完整物品流向圖

```
探索
  │
  ├─ 種子 (type:種子)
  │     └─→ 農女 [耕種] 委託（單物品配方 + 倒數）
  │               └─→ 作物 / 食品 (type:食材/食品)
  │                         ├─→ 送禮 → 好感↑
  │                         └─→ 自用 → 探索 BUFF
  │
  ├─ 稀有食材 (type:食材)
  │     └─→ 送禮 或 自用 BUFF（直接使用）
  │
  ├─ 煉金素材 (type:素材)
  │     └─→ 魔女 [煉製] 委託（單物品配方 + 倒數）
  │               └─→ 禮物 (type:禮物)（藥水/寶石/金工品）
  │                         └─→ 送禮 → 好感↑
  │
  ├─ 擴建物資 (type:素材，具體種類 TBD)
  │     └─→ 村長夫人 [倉庫] → 擴建
  │               └─→ 倉庫 +50 格
  │
  ├─ （守衛 [探索周圍] 委託，空手可派遣）
  │     └─→ 基礎物資（種子 / 初階素材）
  │               └─→ 作為農女/魔女委託的輸入來源（循環）
  │
  └─ 配方卷軸 (type:其他)
        └─→ 解鎖新配方（加入對應角色的配方表）TBD

好感度累積途徑：送禮 + 對話問答（玩家發問 40 題）+ 紅點角色發問（70 題）
  └─→ 達到門檻 → 觸發階段躍遷 → CG 劇情 / HCG（里程碑獎勵）

主線任務引導（L3/L4 紅點）貫穿其中，每任務 = 機制/功能的敘事包裝
  └─→ 解鎖新配方 / 新區域 / CG / 倉庫擴建引導
```
