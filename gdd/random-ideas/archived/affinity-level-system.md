# 好感度等級與好感度分離機制

> **建立日期**: 2026-04-16
> **狀態**: ✅ 已定案
> **相關主題**: core-definition.md（好感度系統）、character-interaction.md（互動流程）、各角色 character-spec.md（五階段關係曲線）

## 內容

製作人提出好感度系統的具體結構：

1. **區分「好感度」與「好感度等級」** — 好感度是數值（可累積），好感度等級是階段
2. **升級機制** — 好感度到達當前等級上限時，好感度等級提升，並提升下一等級所需的好感度數值
3. **10% 里程碑對話** — 好感度每滿 10%（當前等級進度）觸發一段小對話，在互動頁面播放
4. **等級提升 CG 劇情** — 好感度等級提升時進入有 CG 的劇情
